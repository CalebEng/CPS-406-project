The instructions to run the online shopping systems are as follows:

(MAKE SURE JAVA IS PROPERLY INSTALLED ON YOUR SYSTEM)

a) you may open "Main.java" and simply run it through VSCode
b) or you may open the command line in the directory and run "javac Main.java && java Main"

- When looking for any users, simply go into "dbUsers.txt" in the "database" folder to view a list of sample users.
	- simply look for the "|AccEmail:" & "|AccPassword:" fields, which will be the email and password you can use to log in as that user.

- When looking at any products, simply go into "dbProducts.txt" in the "database" folder to view a list of sample products.
